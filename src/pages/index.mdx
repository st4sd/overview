## Try it yourself

<Row>
<Column colMd={5} colLg={8}>

`pip install "st4sd-runtime-core[develop]>=2.5.0"`, which installs the command-line-tool `elaunch.py`, and execute the following code to run a toy-experiment:

</Column>

<Column colMd={2} colLg={3} offsetMd={1} offsetLg={1}>
<Aside>
python 3.7 to 3.10 tested
</Aside>
</Column>
</Row>

```bash
git clone https://github.com/st4sd/sum-numbers.git
elaunch.py --nostamp -l40 sum-numbers
```

This clones a repository into a directory called `sum-numbers` in your current working dir.
It then uses the `elaunch.py` tool to run the toy experiment defined in `sum-numbers` directory.

The output of the `elaunch.py` command will be a directory called `sum-numbers.instance` in the same directory you ran `elaunch.py`.

To learn more visit [ST4SD Core](/st4sd-core-getting-started) and try an example of running a real chemistry experiment.
Then check the [ST4SD Core tutorial](/tutorial) which uses the `sum-numbers` workflow to illustrate key features.

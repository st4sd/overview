**ST4SD Cloud** creates, and maintains, robust and secure deployments of ST4SD on OpenShift.
It is built on cutting edge technologies for deploying and managing complex applications.
Anyone with knowledge of Kubernetes/OpenShift administration can use ST4SD Cloud to deploy ST4SD on an OpenShift cluster.

## Try it yourself

<!-- VV: This will have to do till we write a better `quickstart` for ST4SD Cloud -->

Add the **Simulation Toolkit for Scientific Discovery (ST4SD)** operator to the Operator Catalog of your OpenShift cluster:

```
oc apply -f https://raw.githubusercontent.com/st4sd/st4sd-olm/main/examples/deploy.yaml
```

Then follow our <a href="https://github.com/st4sd/st4sd-olm" target="_blank">instructions</a> to configure and install ST4SD:
ST4SD Core, ST4SD Services, ST4SD Registry all managed by ST4SD Cloud.
